# Decision journal — 25


## Decision
The `language_tiers` package does not exist and must be created at `internal/language_tiers/`. It will define: a `Tier` type (int enum) with constants `TierCapable` (has a registered parser/analyzer — go, ts, js, py, rs), `TierGeneric` (in certifiableExts but no analyzer — swift, java, ruby, etc.), and `TierNone` (not recognized). A public function `TierForLanguage(lang string) Tier` will consult `analysis.ForLanguage` to determine capability. The package will be imported by scorer.go.
